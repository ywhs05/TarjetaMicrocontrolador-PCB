# TarjetaMicrocontrolador-PCB

### Descripción
PCB que facilita la conexión con el microcontrolador LAUNCHXL-F28379D.

Proyecto del laboratorio LabCES de la Universidad de Costa Rica.

Permite facilitar la conexión con el microcontrolador LAUNCHXL-F28379D con las otras placas modulares:
- [Tarjeta digital a fibra óptica (DaFO)](https://github.com/ywhs05/DigitalToFiberOpticConverter-PCB)
- [Tarjeta sensor](https://github.com/ywhs05/TarjetaSensor-PCB)

Esta PCB es la tarjeta principal del sistema modular, tendrá la alimentación para las placas y el microcontrolador.
